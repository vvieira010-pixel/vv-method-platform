# MET Proficiency Mastery — Study Overview

## What is the MET?

The Michigan English Test (MET) is a standardized English proficiency exam at B1–C1 CEFR levels, commonly required for healthcare professionals and others needing to demonstrate practical English skills for work, study, or professional licensing.

## Four Competencies Assessed

| Competency | Skills Tracked | Weight |
|---|---|---|
| **I: Listening** | Comprehension of spoken discourse: main idea, detail, inference, attitude, function | 25% |
| **II: Reading + Grammar + Vocabulary** | Reading comprehension with integrated grammar and vocabulary sub-systems | 25% |
| **III: Writing** | Personal responses and formal essay with 7 assessment standards | 25% |
| **IV: Speaking** | 5 structured tasks with execution formulas under timed conditions | 25% |

## CEFR Score Bands (MET 0–80 scale)

| Level | Score Range | Description |
|---|---|---|
| C1 | 64–80 | Advanced; complex language controlled |
| B2 | 53–63 | Upper-intermediate; target for most learners |
| B1 | 40–52 | Intermediate; functional competence |
| A2 | 27–39 | Elementary; basic communication |
| Below A2 | 0–26 | Beginner |

Many healthcare/work visa profiles require overall ≥58, speaking ≥59.

## The Learning Cycle

```
Diagnosis → Homework → Submission → Feedback → Next Diagnosis
```

1. **Diagnosis** — AI analyzes student evidence against MET rubrics, identifies skill gaps
2. **Homework** — Targeted exercises generated from diagnosis priorities
3. **Submission** — Student completes and submits responses (text + audio)
4. **Feedback** — Teacher reviews, corrects, provides gap analysis with next steps
5. **Repeat** — Next session addresses remaining gaps

## Seven Skills Tracked

1. Speaking — 5 tasks (Q1–Q5)
2. Writing — 2 sections (W1: 3 questions, W2: essay)
3. Reading — Multiple passage types
4. Listening — 3 parts (short conversations, long conversations, talks)
5. Grammar — 9 sub-systems
6. Vocabulary — 9 sub-skill areas
7. Test Strategy — Cross-cutting exam technique

## Scoring System

- Each skill scored 0–80 (MET scale)
- Speaking: 3 categories × 0–4 (Task Completion, Language Resources, Intelligibility)
- Writing: 5 categories × 0–4 (Grammatical Accuracy, Vocabulary, Mechanics, Cohesion, Task Completion)
- Diagnostic labels: Secure → Consolidating → Developing → Not yet evident

## Key Connections

### Listening ↔ Speaking
Both test real-time processing. Improvement in one helps the other.
- **Shared skill**: Recognizing discourse signposts (but, however, actually)
- **Shared challenge**: Processing under time pressure
- **Study tip**: Practice with audio → respond verbally in the same session

### Reading ↔ Writing
Both involve text. Strong readers become stronger writers.
- **Shared skill**: Text organization, coherence, main idea development
- **Transfer effect**: Reading models good writing; writing clarifies reading comprehension
- **Study tip**: After reading a passage, write a 1-paragraph summary in your own words

### Grammar ↔ All Skills
Grammar control underpins every competency.
- **Speaking**: Tense consistency in narratives (Q2), conditional structures (Q4 comparison)
- **Writing**: Grammatical Accuracy is 1 of 5 scoring criteria
- **Reading**: Grammar gap-fill section within Reading competency
- **Listening**: Understanding complex structures in talks (Part 3)

### Vocabulary ↔ All Skills
Lexical range determines performance ceiling.
- **Speaking**: Precise vocabulary = higher Language Resources score
- **Writing**: Vocabulary range is an independent scoring criterion
- **Reading**: Vocabulary in context questions
- **Listening**: Recognizing paraphrased vocabulary in distractors
- **Study tip**: Learn vocabulary in topic sets (not isolated words)

### Test Strategy ↔ Every Section
Exam technique separates knowledge from performance.
- **Time management** affects all sections
- **Distractor management** directly impacts Reading and Listening scores
- **Self-correction** improves Speaking and Writing quality

## Scoring Criteria Dependency Map

A score in one area depends on underlying skills:

```
SPEAKING SCORE
 ├─ Task Completion ← Idea Development + Supporting Details
 ├─ Language Resources ← Grammar Accuracy + Vocabulary Range
 └─ Intelligibility ← Pronunciation + Fluency

WRITING SCORE
 ├─ Task Completion ← Direct relevance + Supporting detail
 ├─ Organization ← Paragraph structure + Idea sequencing
 ├─ Development ← Example + Explanation + Elaboration
 ├─ Grammatical Accuracy ← Sentence structure + Tense control
 ├─ Vocabulary Range ← Word choice + Lexical sophistication
 ├─ Coherence ← Logical flow + Transitions
 └─ Register ← Formal/informal appropriateness

READING SCORE
 ├─ Main Idea ← Skimming + Gist recognition
 ├─ Detail ← Scanning + Precise matching
 ├─ Inference ← Text clue + Logical reasoning
 ├─ Vocabulary in Context ← Context clue recognition
 └─ Distractor Resistance ← Paraphrase recognition

LISTENING SCORE
 ├─ Main Idea ← Speaker purpose recognition
 ├─ Detail ← Selective attention
 ├─ Inference ← Context + discourse signposts
 ├─ Distractor Resistance ← Paraphrase detection
 └─ Prediction ← Anticipation + context use
```

## Error Classification System (Gap Analysis)

Heritage (2010) framework used when analyzing student work:

```
STUDENT ERROR
     │
     ├── CONCEPTUAL ──→ Student doesn't understand the idea
     │                      │
     │                      └──→ Requires re-teaching
     │
     ├── PROCEDURAL ──→ Student understands but makes errors
     │                      │
     │                      └──→ Requires targeted practice
     │
     └── COMMUNICATION ──→ Student can do it but can't express it
                            │
                            └──→ Requires modelling of expression
```

This classification determines the correct teaching response. Conceptual gaps need re-teaching. Procedural gaps need targeted practice. Communication gaps need modeling.

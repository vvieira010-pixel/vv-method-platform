# MET Exam — Key Terms and Definitions

## Core Exam Concepts

| Term | Definition |
|---|---|
| **MET** | Michigan English Test — standardized English proficiency exam at B1–C1 CEFR levels |
| **CEFR** | Common European Framework of Reference for Languages — international standard for describing language ability (A1 → C2) |
| **B1** | Intermediate level — can handle everyday situations, express opinions, describe experiences |
| **B2** | Upper-intermediate level — target for most learners; can interact fluently, argue a point, write detailed text |
| **C1** | Advanced level — can use language flexibly and effectively for social, academic, and professional purposes |

## Scoring Terms

| Term | Definition |
|---|---|
| **0–80 Scale** | MET overall score range |
| **Score Confidence Level** | How reliable a score estimate is, based on amount of evidence available |
| **Diagnostic estimate** | Confident enough to guide teaching from available evidence |
| **Provisional estimate** | Some evidence but needs more before confident |
| **Mock-test estimate** | Full simulation test completed |
| **Secure** | Consistent performance — a strength |
| **Consolidating** | Inconsistent — needs repetition and cuing |
| **Developing** | Partial control only in favorable conditions |
| **Not yet evident** | Gap confirmed — cannot yet produce reliably |

## Speaking Terms

| Term | Definition |
|---|---|
| **Task Completion** | How well the speaker addressed the task — relevance, quantity, elaboration |
| **Language Resources** | Vocabulary range, grammar accuracy/complexity, fluency |
| **Intelligibility / Delivery** | Pronunciation, rhythm, hesitation, clarity of speech |
| **Register** | Level of formality — formal vs. informal language |
| **Neutral register** | Balanced, objective language used for advantages/disadvantages (Q4) |
| **Committed register** | Strong, opinionated language used for persuasion (Q3, Q5) |
| **Execution Formula** | Step-by-step structure for each speaking task type |
| **Spatial coverage** | Describing all areas of a picture (foreground, background, centre, etc.) |
| **Narrative arc** | Story structure: setup → event → outcome |

## Writing Terms

| Term | Definition |
|---|---|
| **Grammatical Accuracy** | Correct use of grammar — error frequency and severity |
| **Mechanics** | Sentence boundaries, punctuation, spelling |
| **Cohesion** | How ideas are linked together using connective devices |
| **Coherence** | Logical flow and clarity of ideas |
| **Register Consistency** | Maintaining appropriate formality throughout |
| **Expansion Frame** | Opinion → Reason → Support (the basic argument structure) |

## Reading Terms

| Term | Definition |
|---|---|
| **Main idea** | The central point or gist of a text |
| **Detail** | Specific information stated in the text |
| **Inference** | A logical conclusion based on text evidence + reasoning |
| **Paraphrase recognition** | Identifying the same idea expressed in different words |
| **Distractor** | A tempting wrong answer choice |
| **Lexical distractor** | A wrong answer that reuses exact words from the text/audio |
| **Scanning** | Quickly looking for specific information |
| **Skimming** | Quickly reading for the general idea |

## Listening Terms

| Term | Definition |
|---|---|
| **Mid-stream information shift** | When a speaker changes their mind or corrects themselves mid-conversation |
| **Discourse signpost** | Words that signal the structure of what's being said (but, however, actually, first, then) |
| **Note-taking overload** | Trying to write everything down, which causes you to miss what comes next |
| **Prediction anticipation** | Using context to predict what a speaker will say next |

## Grammar Terms

| Term | Definition |
|---|---|
| **Present Simple** | Used for habits, routines, general truths ("I usually have breakfast at 8am") |
| **Present Continuous** | Used for temporary situations around now ("This week I am staying at a hotel") |
| **Past Simple** | Completed actions in the past ("I arrived yesterday") |
| **Past Continuous** | Background actions in progress when something happened ("I was walking when it started to rain") |
| **Present Perfect** | Past actions connected to now ("I have lived here for three years") |
| **First Conditional** | Real future possibility ("If it rains, I will stay home") |
| **Second Conditional** | Unreal/hypothetical present ("If I had more time, I would travel") |
| **Third Conditional** | Impossible past condition ("If I had studied, I would have passed") |
| **Passive Voice** | Object becomes subject focus ("The dish was prepared by the chef") |
| **Reported Speech** | Reporting what someone said, with tense backshift ("She said she was tired") |
| **Relative Clause** | A clause that modifies a noun ("The nurse who helped me was kind") |

## Test Strategy Terms

| Term | Definition |
|---|---|
| **Time management** | Pacing yourself across sections and questions |
| **Question type recognition** | Identifying what each question is actually asking |
| **Distractor management** | Avoiding tempting but incorrect answers |
| **Evidence selection** | Choosing the right evidence from text/audio to support your answer |
| **Self-correction** | Noticing and fixing your own errors |
| **Exam anxiety control** | Managing stress during the test |

## Platform-Specific Terms

| Term | Definition |
|---|---|
| **Diagnosis** | AI-generated analysis of student performance against MET rubrics |
| **Homework** | Targeted exercises generated from diagnosis priorities |
| **Submission** | Student's completed homework responses |
| **Review** | Teacher's correction and feedback on a submission |
| **Error Bank** | Logged error patterns for a student, categorized and tracked over time |
| **Gap Analysis** | Systematic identification of gaps classified as Conceptual / Procedural / Communication |
| **Spaced Repetition** | Review scheduling at increasing intervals [1, 3, 7, 14, 30 days] |
| **SEEDS** | 5-stage regenerative inquiry: Sense → Envision → Experiment → Design to Last → Share |
